# Simplistic Portfolio (Simpleton)

This is a simplistic portfolio built using React. It features a clean design with a black background and quick link buttons to important resources such as your resume, current job, LinkedIn, and GitHub.

## Project Structure

```
simplistic-portfolio
├── src
│   ├── index.tsx
│   ├── App.tsx
│   ├── components
│   │   └── QuickLinks.tsx
│   ├── styles
│   │   └── App.css
│   └── types
│       └── index.d.ts
├── public
│   └── index.html
├── package.json
├── tsconfig.json
└── README.md
```

## Getting Started

To get started with this project, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd simplistic-portfolio
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the application:**
   ```
   npm start
   ```

4. Open your browser and navigate to `http://localhost:3000` to view your portfolio.

## Features

- Black background for a minimalist look.
- Quick links to your resume, current job, LinkedIn, and GitHub.
- Built with React and TypeScript for type safety.

## Customization

You can customize the links in the `QuickLinks.tsx` component to point to your actual resources. Additionally, you can modify the styles in `App.css` to change the appearance of the buttons or the overall layout.
